Senior Project

Real Time Vehicle Tracking and Classification

Northrop Grumman sponsored this project to look at time-dependent classification of vehicles. The goal of this group project was to create a real-time model that could take in video of vehicles and classify which type of vehicle it is. This project was more of a research project to look at which machine learning models performed the best given the constraints.
